**택스몬스터 (SWA/SA & DevOps Engineer) | 2024.10 ~ 현재**[cite: 1]

* **현지 기술 내재화 기여:** 탄자니아 현지 ICT 운영팀을 대상으로 GitLab 운영 및 CI/CD 파이프라인 기술 내재화 교육 기획 및 진행[cite: 1].
